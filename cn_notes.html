Computer Networks - Unit 1

Course Objectives

  • Understand computer networks basics.
  • Identify different network devices and protocols.
  • Recognize network topologies and their associated protocols.
  • Understand routing, IP addressing, and transport protocols.
  • Apply various application protocols.

Reference Books

  • "Computer Networks" - Tanenbaum, PHI Learning
  • "Data Communication & Networks", Forouzan, TMH
  • "TCP/IP-Protocol Suite", Forouzan, TMH
  • "Computer Networks and Internets", Comer, Pearson

Course Content

  • Basic Concepts of Networking
  • Uses of Networks
  • Types of Networks (LAN, MAN, WAN, etc.)
  • Networking Devices (Router, Switch, Hub, etc.)
  • Network Media (Wired & Wireless)
  • Network Topologies (Bus, Star, Mesh, etc.)

Uses of Networks

  • Simultaneous access to data
  • Shared Peripheral Devices
  • Personal Communication (VoIP, Video Conferencing)
  • Data Backup and Security

Types of Networks

  • LAN (Local Area Network)
  • MAN (Metropolitan Area Network)
  • WAN (Wide Area Network)
  • PAN (Personal Area Network)
  • VPN (Virtual Private Network)

Networking Devices

  • NIC Card
  • Repeater
  • Hub
  • Switch
  • Router
  • Firewall

Network Topologies

  • Bus
  • Ring
  • Star
  • Mesh
  • Tree
  • Hybrid

Wireless Networks

Wireless networks use radio signals, microwaves, and infrared to transmit data.

  • Wi-Fi
  • Bluetooth
  • Mobile Networks (3G, 4G, 5G)
